# Tic-Tac-Toe Web App

A clean, modern Tic-Tac-Toe game built with **Node.js**, **HTML5**, **CSS3**, and **Vanilla JavaScript**. This project uses no external dependencies (using Node's built-in `http` module) for a lightweight and easy-to-run experience.

## 🚀 Features
- **Modern UI:** Responsive design with a clean, centered layout.
- **Dynamic Board:** Visual indicators for turns, wins, and draws.
- **2-Player Mode:** Alternate between Player X and Player O.
- **Reset Functionality:** Quickly restart the game at any time.

## 🛠️ How to Run Locally

1. **Prerequisites:** Make sure you have [Node.js](https://nodejs.org/) installed on your machine.
2. **Clone the project:** (Or just copy the files).
3. **Start the server:**
   ```bash
   node server.js
   ```
4. **Play:** Open your browser and navigate to `http://localhost:3000`.

## 📂 Project Structure
- `server.js`: Lightweight Node.js server.
- `public/`: Contains the frontend assets (HTML, CSS, JS).
