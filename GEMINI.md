# Project Context: Tic-Tac-Toe Web App

## Overview
A lightweight, dependency-free Tic-Tac-Toe web application built with a Node.js backend and a Vanilla JavaScript frontend.

## Technical Stack
- **Backend:** Node.js (utilizing the built-in `http` module).
- **Frontend:** HTML5, CSS3, Vanilla JavaScript.
- **Server:** `server.js` (static file server).

## Project Structure
- `server.js`: The main entry point that serves files from the `public/` directory.
- `public/`:
  - `index.html`: The game UI.
  - `style.css`: The application's styling.
  - `script.js`: The game logic and UI interactions.

## Operational Instructions
- **Start Server:** Run `node server.js`.
- **Access App:** The server runs on `http://localhost:3000` by default.
